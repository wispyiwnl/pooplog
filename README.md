# 💩 PoopLog

Rastreador personal de deposiciones basado en la escala de Bristol. Registra, visualiza y lleva historial de tu actividad intestinal — con calendario mensual, score digestivo y soporte para registros tardíos.

Proyecto de aprendizaje y uso personal, construido junto a [Claude](https://claude.ai) (Anthropic).

> **Esta app no reemplaza criterio médico.** Si tienes dudas sobre tu salud digestiva, consulta a un profesional.

## Stack
HTML · CSS · JavaScript · Supabase · Netlify
